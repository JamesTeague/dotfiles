## 5. `home/dot_zshrc.tmpl` lines 30-80 — loader logic (verbatim quote)

**File total:** 105 lines. The loader spans lines 32-77.

**Three-pass load order (verbatim relevant lines):**

```zsh
# all of our zsh files
typeset -U config_files
config_files=({{.chezmoi.homeDir}}/.topics/*/*.zsh)

# load the path files
for file in ${(M)config_files:#*/path.zsh}
do
  source $file
done


autoload -U compinit add-zsh-hook
compinit -u add-zsh-hook chpwd
compaudit | xargs chmod g-w,o-w

# load every completion after autocomplete loads
for file in ${(M)config_files:#*/completion.zsh}
do
  source $file
done

# ...fzf helper block...

# load everything but the path and completion files
for file in ${${config_files:#*/path.zsh}:#*/completion.zsh}
do
  source $file
done

unset config_files
```

**Three loader passes (in order):**

1. **PASS 1 (path):** `${(M)config_files:#*/path.zsh}` — sources every `*/path.zsh` first.
2. **`compinit`** runs between pass 1 and pass 2 (autoload tab-completion engine).
3. **PASS 2 (completion):** `${(M)config_files:#*/completion.zsh}` — sources every `*/completion.zsh` AFTER `compinit`.
4. **PASS 3 (middle / everything else):** `${${config_files:#*/path.zsh}:#*/completion.zsh}` — sources every other `*.zsh` (aliases, config, eval).

**WHY load order matters:**
- `path.zsh` must run FIRST so other tools' `eval $(tool init)` invocations see the right `$PATH`.
- `completion.zsh` must run AFTER `compinit` so completion functions register against an initialized completion engine.
- Middle pass runs last so it sees the populated PATH and the live completion engine.

**Destination-side note:** chezmoi strips the `dot_` prefix on apply, so `dot_topics/` lands at `~/.topics/` (lowercase, no underscore). The glob in the loader matches `~/.topics/*/*.zsh` — confirmed against the dest-rendered loader.

**install.sh handling (cross-link to inspection 1):** `install.sh` files end in `.sh` (NOT `.zsh`), so the loader does NOT pick them up. Instead they are discovered + executed by `run_onchange_after_03-install-topics.sh.tmpl`:

```bash
find {{.chezmoi.homeDir}}/.topics -name install.sh | while read installer ; do /bin/bash -c "${installer}" ; done
```
