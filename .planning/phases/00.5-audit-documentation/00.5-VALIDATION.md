## Sampling Rate

- **After every task commit:** Run `bash .planning/phases/00.5-audit-documentation/checks/quick.sh` (file-existence + grep checks for the artifact the task produced)
- **After every plan wave:** Run `bash .planning/phases/00.5-audit-documentation/checks/full.sh` (quick + `chezmoi diff -x externals` on Mac personal + `chezmoi execute-template` rendering)
- **Before `/gsd:verify-work`:** Full suite green on Mac personal AND Teague has run `chezmoi diff -x externals` on Mac work with empty output
- **Max feedback latency:** ~15 seconds for automated; human Mac work gate is async (Teague-driven)
